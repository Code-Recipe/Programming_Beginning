目录
------
1. 初入编程

  * 练习环境小热身
  * 计算
  * 小练习
2. 选择判断
  * 运行顺序
  * 判断是不是
  * 如果…否则…
  * 更多种情况——否则如果
  * 且与或
  * 偷懒的计算机——最小化求值原理
  * 不是的情况——“非”运算

3. 变量和类型
